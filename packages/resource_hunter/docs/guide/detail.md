# 详情接口

### 请求参数

```json5
{
  "sid": "",       // 作品ID，可以为空
  "vid": "",       // 视频ID
  "link": ""       // 视频地址
}
```

### 响应结果

```json5
{
  "sid": "",       // 作品ID，可以为空
  "vid": "",       // 视频ID
  "title": "",     // 标题
  "metas": [""],   // 更多信息
  "desc": "",      // 描述
  "cht": "",       // 横屏封面
  "cvt": "",       // 竖屏封面
  "link": ""       // 视频地址
}
```
